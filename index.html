Теория графов — Полное руководство

Основные понятия теории графов

1. Что такое граф?

Граф — это математическая структура, состоящая из:

  • Вершин (узлов) — основных элементов графа
  • Рёбер — связей между вершинами

Пример: В социальной сети люди — это вершины, а дружеские связи — рёбра.

2. Виды графов

Неориентированные графы:

  • Рёбра не имеют направления
  • Ребро (A,B) = ребру (B,A)
  • Пример: Дороги между городами

Ориентированные графы (орграфы):

  • Рёбра имеют направление
  • Дуга A→B не то же самое, что B→A
  • Пример: Односторонние улицы

Взвешенные графы:

  • Каждому ребру присвоен вес (число)
  • Пример: Расстояние между городами

3. Основные характеристики графов

Степень вершины — количество рёбер, инцидентных вершине:

  • Для орграфов: полустепень исхода и захода
  • Изолированная вершина — степень 0
  • Висячая вершина — степень 1

Путь — последовательность вершин, где каждые две соседние соединены ребром:

  • Простой путь — без повторения вершин
  • Цикл — путь, где начальная и конечная вершины совпадают

Важные алгоритмы и понятия

1. Обход графа

Поиск в ширину (BFS):

  1. Начинаем с начальной вершины
  2. Посещаем все соседние вершины
  3. Повторяем для каждой следующей "волны" вершин

Поиск в глубину (DFS):

  1. Начинаем с начальной вершины
  2. Идём "вглубь" по одному направлению
  3. Возвращаемся, когда дальше идти некуда

2. Связность графа

Связный граф — между любыми двумя вершинами существует путь.

Компонента связности — максимальный связный подграф.

Пример: В графе друзей из разных стран могут быть отдельные компоненты связности для каждой страны.

3. Деревья

Дерево — связный граф без циклов.

Свойства деревьев:

  • Между любыми двумя вершинами ровно один путь
  • Число рёбер = число вершин - 1
  • Добавление любого нового ребра создаёт цикл

4. Кратчайшие пути

Алгоритм Дейкстры — находит кратчайшие пути от одной вершины до всех остальных.

Условия:

  • Граф должен быть взвешенным
  • Веса рёбер должны быть неотрицательными

5. Двудольные графы

Двудольный граф — вершины можно разделить на два множества так, что рёбра соединяют только вершины из разных множеств.

Критерий двудольности:

  • Граф двудольный ⇔ не содержит циклов нечётной длины
  • Можно проверить раскраской в два цвета
